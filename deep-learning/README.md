# Deep Learning

Раздел с учебными ноутбуками по deep learning.

## Состав

- `notebooks/mlp_mnist_pytorch.ipynb` — полносвязная сеть на PyTorch для распознавания рукописных цифр.
- `notebooks/multiclass_semantic_segmentation.ipynb` — задача трёхклассовой семантической сегментации.
- `notebooks/named_entity_recognition_transformers.ipynb` — задача NER с использованием современных NLP-инструментов.

## Как оформить этот раздел ещё сильнее

- добавить краткие итоги по каждому ноутбуку;
- вынести картинки/предсказания в `assets/`;
- прописать используемые датасеты и основные метрики.
