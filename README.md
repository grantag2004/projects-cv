Этот репозиторий содержит выполненные задания и учебные проекты в рамках Т-Академии по направлению «Аналитика», а также работы, выполненные в университете по курсам deep learning, высокопроизводительная и параллельная обработка данных, курсовая работа по тематике DL.

В репозитории собраны материалы по двум крупным направлениям. Первая часть — задания Т-Академии по аналитике: теория вероятностей и математическая статистика, A/B-тесты и основы машинного обучения. В этих ноутбуках рассматриваются распределения случайных величин, t-критерии, p-value, мощность тестов и дизайн экспериментов, а также базовые и прикладные ML-задачи: KNN, линейная и логистическая регрессия, регуляризация, ансамбли, бустинг, предобработка данных и анализ важности признаков.

Вторая часть — университетские работы. Сюда входят ноутбуки по deep learning, мини-проект по распределённой обработке текстовых данных и курсовая работа по распределённому обучению свёрточных нейронных сетей на GPU.

Кратко о содержимом репозитория

Т-Академия · Аналитика
Ноутбуки этого блока посвящены задачам по теории вероятностей, математической статистике и A/B-тестам, а также основам машинного обучения. В работах реализованы и разобраны визуальный анализ распределений, критерий Уэлча, моделирование распределения t-статистики, оценка мощности тестов и расчёт параметров эксперимента. В ML-части собраны задачи по KNN, линейной и логистической регрессии, регуляризации, классификации мошеннических операций, деревьям решений, ансамблям, бустингу, предобработке признаков и интерпретации моделей.

Университет · Deep Learning
В ноутбуках по DL представлены задания по реализации и обучению полносвязных сетей в PyTorch, задаче трёхклассовой семантической сегментации и задаче Named Entity Recognition. В NER-части есть подготовка данных, BiLSTM-модель и transformer-подход; в задаче сегментации — подготовка датасета, метрики и PSPNet-подобная архитектура.

Университет · Мини-распределённая система
Отдельный проект — мини-система для распределённой обработки текстовых документов. В ней реализованы клиентская консоль, менеджеры и воркеры, взаимодействующие через RabbitMQ, выбор лидера и heartbeat-механизм между менеджерами, репликация данных на резервные узлы, загрузка текстовых файлов, поиск по слову и вычисление TF-IDF через MapReduce для поиска похожих документов.

Университет · Курсовая работа
Курсовая посвящена исследованию методов распределённого обучения свёрточных нейронных сетей с использованием графических ускорителей. В работе рассмотрены стратегии параллелизма, а в практической части реализованы и сравнены DDP, pipeline parallelism и tensor parallelism для ResNet-152 на датасете Tiny-ImageNet-200 на узле МГУ-270. В выводах показано, что DDP даёт лучшую масштабируемость по времени, pipeline-подход чувствителен к числу микробатчей, а tensor/hybrid-подходы помогают по памяти, но упираются в коммуникационный оверхед.
