# Mini Distributed Search System

Мини-распределённая система поиска по текстовым документам на Python.

## Идея проекта

Система поднимает несколько manager- и worker-процессов, распределяет документы между worker-узлами, хранит резервные копии у replica-воркеров и поддерживает простой поиск по словам и по смысловой близости через TF-IDF.

## Возможности

- manager / worker архитектура;
- запуск резервных менеджеров;
- election лидера;
- primary / replica worker-модель;
- загрузка документов с равномерным и неравномерным распределением;
- поиск по слову;
- TF-IDF и поиск похожих документов.

## Файлы

- `client.py` — CLI-клиент и команды управления системой;
- `manager.py` — управляющий узел, leader election, orchestration;
- `worker.py` — хранение документов, обработка запросов, MapReduce-этапы;
- `common.py` — общие константы, очереди, обработка текста.

## Зависимости

```bash
pip install -r requirements.txt
```

Также нужен запущенный RabbitMQ.

## Быстрый запуск

```bash
python client.py
```

Дальше в консоли:

```text
start 2 1
load ./texts e
find transformer
find_like distributed training on gpu
```
