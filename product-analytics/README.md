# Product Analytics

Работы по продуктовой аналитике, математической статистике и основам машинного обучения.

## Подразделы

### Probability and Statistics
Ноутбуки по распределениям, t-test, оценке мощности тестов и статистическому анализу.

### Machine Learning Basics
Ноутбуки по KNN, линейным моделям, регуляризации, деревьям решений, ансамблям, бустингу, preprocessing и feature importance.
