# Distributed CNN Training Coursework

Курсовая работа, посвящённая исследованию методов обучения свёрточных нейронных сетей с использованием графических ускорителей.

## Содержание

В работе рассматриваются:
- параллелизм данных;
- конвейерный параллелизм;
- тензорный параллелизм;
- сравнительный анализ стратегий на GPU.

## Материалы

- `report/distributed_cnn_training_coursework.pdf` — полный текст курсовой работы.

## Почему это сильный проект для GitHub

- есть исследовательская постановка;
- есть обзор подходов и практические эксперименты;
- тема лежит на стыке deep learning и HPC;
- проект хорошо показывает навыки анализа, экспериментов и инженерной реализации.
