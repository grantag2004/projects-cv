# Publish Checklist

## Перед публикацией

- [ ] Проверить, нет ли в ноутбуках лишних output-ячееек с персональными путями.
- [ ] Убедиться, что внутри ноутбуков нет токенов, ключей и приватных ссылок.
- [ ] При необходимости очистить outputs через Jupyter / nbconvert.
- [ ] Добавить 2–4 скриншота графиков или результатов.
- [ ] Заполнить раздел About на GitHub.
- [ ] Закрепить репозиторий в профиле GitHub.

## После публикации

- [ ] Добавить pinned repositories.
- [ ] Сделать отдельные репозитории для самых сильных проектов.
- [ ] Добавить английскую версию README при необходимости.
